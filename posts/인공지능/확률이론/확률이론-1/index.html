    нҷ•лҘ мқҙлЎ  (1)

    нҷ•лҘ ліҖмҲҳлһҖ?

    нҷ•лҘ ліҖмҲҳ X лҠ” н‘ңліёмқҳ м§‘н•© S мқҳ мӣҗмҶҢ eлҘј мӢӨмҲҳк°’ X(e) = xм—җ лҢҖмқ‘мӢңнӮӨлҠ” н•ЁмҲҳмһ…лӢҲлӢӨ.

    мҳҲлҘјл“Өм–ҙ, 6л©ҙмІҙ мЈјмӮ¬мң„м—җм„ң ‘2’к°Җ лӮҳмҳ¬ нҷ•лҘ мқҖ 1/6мқҙ лҗ©лӢҲлӢӨ. мқҙ л•Ң, нҷ•лҘ ліҖмҲҳ X(2) лҠ” 1/6мһ…лӢҲлӢӨ.

    л¬јлЎ , X(1) = X(2) = … = X(6) = 1/6мһ…лӢҲлӢӨ.

    м—°мҶҚнҷ•лҘ ліҖмҲҳ

    м „м„ёкі„ мӮ¬лһҢл“Өмқҳ нӮӨ лҚ°мқҙн„°лҘј к°Җм§Җкі  нӮӨк°Җ 170cmмқј нҷ•лҘ мқ„ кө¬н•ңлӢӨкі  н•ҙлҙ…мӢңлӢӨ.

    к·ёлҹ°лҚ° нӮӨк°Җ м •нҷ•нһҲ 170.00…cmмқё мӮ¬лһҢмқҖ мЎҙмһ¬н•ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ 0%м—җ мҲҳл ҙн•©лӢҲлӢӨ.

    мқҙл•Ң лҲ„м Ғл¶„нҸ¬н•ЁмҲҳлҘј мӮ¬мҡ©н•ҳм—¬ нҷ•лҘ ліҖмҲҳлҘј н‘ңнҳ„н•ҳкё°лҸ„ н•©лӢҲлӢӨ.

    л§Ңм•Ҫ нҷ•лҘ ліҖмҲҳ X лҘј м•„лһҳмқҳ F(x) мІҳлҹј лӮҳнғҖлӮј мҲҳ мһҲлӢӨл©ҙ, X лҘј м—°мҶҚнҷ•лҘ ліҖмҲҳлқјкі  л¶ҖлҘҙкі , f(x) лҘј X мқҳ нҷ•лҘ л°ҖлҸ„н•ЁмҲҳ(pdf)лқјкі  л¶ҖлҘёлӢӨ.

    лҲ„м Ғл¶„нҸ¬н•ЁмҲҳ (CDF. Cumulative Distribution Function)

    лҲ„м Ғл¶„нҸ¬н•ЁмҲҳлһҖ, мЈјм–ҙм§„ нҷ•лҘ ліҖмҲҳк°Җ нҠ№м • к°’ліҙлӢӨ мһ‘кұ°лӮҳ к°ҷмқҖ нҷ•лҘ мқ„ лӮҳнғҖлӮҙлҠ” н•ЁмҲҳмһ…лӢҲлӢӨ.

    нҷ•лҘ ліҖмҲҳмқҳ н•ЁмҲҳ

    kм°Ёмӣҗмқҳ нҷ•лҘ ліҖмҲҳ лІЎн„° x = (x1, x2, …, xk)к°Җ мЎҙмһ¬н•  л•Ң, yi = gi(x)мқё y = (y1, y2, …, yk)к°Җ мһҲлӢӨкі  н•©мӢңлӢӨ.

    мқҙ л•Ң yмқҳ кІ°н•©нҷ•лҘ л°ҖлҸ„н•ЁмҲҳлҠ” м•„лһҳмҷҖ к°ҷмҠөлӢҲлӢӨ.

    мҳҲм ң

    нҡҢм „мўҢн‘ңкі„лҘј мқҙмҡ©н•ҳм—¬ л°ҳкІҪмқҙ rмқё мӣҗ м•Ҳм—җ лһңлҚӨн•ҳкІҢ м җл“Өмқ„ м°ҚлҠ” н”„лЎңк·ёлһЁмқ„ л§Ңл“Өм–ҙлҙ…мӢңлӢӨ.

    мһ…л ҘліҖмҲҳлҠ” thetaмҷҖ distanceмһ…лӢҲлӢӨ.

    import matplotlib.pyplot as plt
import random
import math


plt.figure()
x = []
y = []
cnt = 0
radius = 5
while cnt <= 10000:
    theta = 2 * math.pi * random.random()
    distance = radius * random.random()
    x.append(distance * math.sin(theta))
    y.append(distance * math.cos(theta))
    cnt += 1
plt.scatter(x, y, s=5)
plt.show()

    thetaлҠ” лһңлҚӨн•ЁмҲҳлҘј мқҙмҡ©н•ҳм—¬ к°„лӢЁнһҲ кө¬н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    distanceлҠ” лһңлҚӨн•ҳкІҢ кө¬н•ҙлІ„лҰ¬л©ҙ к°ҖмҡҙлҚ° м җл“Өмқҙ лӘ°л Өм„ң м°ҚнһҲлҠ” нҳ„мғҒмқҙ л°ңмғқн•©лӢҲлӢӨ.

    л”°лқјм„ң, м җл“Өмқ„ к· л“ұнһҲ л¶„нҸ¬мӢңнӮӨкё° мң„н•ҙ кІ°н•©нҷ•лҘ л°ҖлҸ„н•ЁмҲҳлҘј кө¬н•ҙм•ј н•©лӢҲлӢӨ.

    мһ„мқҳмқҳ distanceм—җ лҢҖн•ң мӣҗмқ„ к·ёлҰ¬кі , к·ё мӣҗ м•Ҳм—җ м җмқҙ л“Өм–ҙк°Ҳ нҷ•лҘ мқ„ кө¬н•ҳл©ҙ,

    F(d) = d^2 / r^2

    F мқҳ м—ӯн•ЁмҲҳлҘј кө¬н•ҳл©ҙ, F(u) = r * sqrt(u)мһ…лӢҲлӢӨ.

    мқҙлҘј л°ҳмҳҒн•ҳм—¬ мҪ”л“ңлҘј мҲҳм •н•ҳл©ҙ,

    import matplotlib.pyplot as plt
import random
import math


plt.figure()
x = []
y = []
cnt = 0
radius = 5
while cnt <= 10000:
    theta = 2 * math.pi * random.random()
    distance = radius * (random.random()) ** 0.5
    x.append(distance * math.sin(theta))
    y.append(distance * math.cos(theta))
    cnt += 1
plt.scatter(x, y, s=5)
plt.show()

    к· мқјн•ҳкІҢ л¶„нҸ¬лҗҳлҠ” кІғмқ„ ліј мҲҳ мһҲмҠөлӢҲлӢӨ.
